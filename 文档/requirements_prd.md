## 一、核心页面需求

### 1. 首页（Home）✅ 已完成

| 需求项 | 详细描述 | 完成状态 |
|---|---|---|
| 顶部导航栏 | 汉堡图标 + "美食图鉴" + 用户头像 | ✅ 实现 |
| 状态栏适配 | 内容不遮挡手机时间/信号状态栏 | ✅ 修复 (`statusBarsPadding`) |
| 搜索栏 | 圆角搜索框，提示文字"搜索我的私厨" | ✅ 实现（UI层，暂无功能） |
| 分类筛选 | 全部/肉禽/蔬菜/汤品 横向可滚动，选中高亮褐色 | ✅ 实现（可交互筛选） |
| 每日灵感卡 | 绿色背景，标题"今天做什么？"，副文案，"随机推荐"按钮 | ✅ 实现（按钮暂无逻辑） |
| 菜品网格 | 2 列卡片网格，每卡有图、分类角标、名称、时间 | ✅ 实现 |
| 底部导航 | Home / Discover / History / Profile 四图标 | ✅ 实现 |
| 浮动操作 | 右下角橙色 `+` 按钮 | ✅ 实现（暂无跳转） |

## 二、技术限制与约定

| 项目 | 规范 |
|---|---|
| 最低系统版本 | Android 7.0 (API 24) |
| 目标编译版本 | API 36 |
| 语言 | 100% Kotlin |
| UI 框架 | Jetpack Compose + Material 3 |
| 多模块结构 | `:app`, `:core:designsystem`, `:feature:home` |
| 数据层 | 目前使用 `fakeRecipes`；后续接入 Room DB |
| 图片加载 | Coil 2.7.0 |
| 导航 | Jetpack Navigation Compose（Phase 5 引入）|
